# ⚽ Scout Latino

Plataforma profesional de scouting para el fútbol amateur latinoamericano.

## 🚀 Despliegue en 4 pasos

### Paso 1 — Instalar Node.js
Descarga desde https://nodejs.org (botón verde LTS)

### Paso 2 — Subir a GitHub
1. Crea cuenta en github.com
2. Descarga GitHub Desktop desde desktop.github.com
3. Abre GitHub Desktop → File → Add local repository → selecciona esta carpeta
4. Commit y Push

### Paso 3 — Publicar en Vercel
1. Crea cuenta en vercel.com con tu cuenta de GitHub
2. New Project → Import desde GitHub → selecciona "scout-latino"
3. Vercel detecta Vite automáticamente → clic en Deploy
4. ¡Tu web está en línea en 2 minutos!

### Paso 4 — Dominio propio (opcional)
1. Compra dominio en namecheap.com (~$12 USD/año)
2. En Vercel: Settings → Domains → Add domain
3. Sigue las instrucciones de DNS

## 💻 Desarrollo local

```bash
npm install
npm run dev
```

Abre http://localhost:5173

## 📦 Construir para producción

```bash
npm run build
```

## 🛠 Stack tecnológico

- **React 18** + **Vite** — Frontend
- **Recharts** — Gráficos y radar charts
- **localStorage** — Almacenamiento local (v1)
- **Claude AI API** — Análisis inteligente
- **Vercel** — Hosting gratuito
